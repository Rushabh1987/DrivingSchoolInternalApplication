# UI Flows

## Admin Sign In

1. Open the app.
2. Enter the admin username and password.
3. The dashboard opens after successful sign in.
4. The session expires after 12 hours or when the browser tab is closed.

## Register Student

1. Click `Add Student` from the header or dashboard quick actions.
2. Fill required fields: student name, phone number, course type, joining date, and status.
3. Fill optional details if available: total fees, alternate phone, email, address, date of birth, learner permit details, license number, and notes.
4. Click `Save Student`.
5. If the phone number already exists, the app shows a duplicate warning.
6. If saving succeeds, the dashboard opens and shows the student in `Recently Added`.
7. The student is also visible in the `Students` view.
